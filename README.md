# Arduino Uno IoT Projects

A collection of IoT projects built with Arduino Uno. This repository curates different IoT projects, each in its own folder with documentation, source code, and wiring guides.

## Projects

| Project | Description |
|---------|-------------|
| [Weather Station](Weather%20Station/) | Temperature and humidity monitor using a DHT11 sensor and 16x2 LCD display |
| [RFID Door Lock](RFID_Door_Lock/) | Contactless access control system using an RC522 RFID reader with LED feedback |

## Getting Started

Each project folder contains its own README with a full component list, wiring diagram, and setup instructions. Browse the folders above to explore.
